This program simulates the Montecarlo's pi number approximation,the original script was programmed in Python and then adapted into an HTML code to make it more interactive.
The original Python script I made uses the relationship between a circle inscribed into a square. The math behind is simple. Knowing that the radius of the inscribed circle is |+-0,5| (r) and a side of the square is 2r*2r = 2r^2 ->
we know that the number of darts that landed in the circle divided the total number of darts throwed is the same that the area of the circle pi*r^2 / 4r^2 = pi/4 => 4*inside darts/total darts is approximately equal to pi.

About the code, the script uses NumPy to work easily and effortless with huge number of darts to throw
I also used matplotlib to render the graph and dots and the random library to allow choosing 2 random numbers between 1 and -1 (dart landing location)
